## Requirements

[Node.js](https://nodejs.org) and [git](https://git-scm.com/)

## Install

Install with [git](https://git-scm.com/)

```sh
$ git clone https://github.com/constantinDev/rbpinger && cd rbpinger
```

## Installing required modules

```sh
$ npm i
```

## Usage

```sh
$ node main --help
```
